About Us

The integrated system consists of beacon-based devices, a mobile application, and cloud services, all working together to provide accurate location tracking and path generation capabilities.

The present invention introduces an integrated system comprising beacon-based devices, a mobile application, and cloud services to achieve precise location tracking and path generation. Utilizing Bluetooth Low Energy (BLE) technology in ESP32 microcontrollers, each beacon device is uniquely identified, facilitating detection and positioning within the designated area. The associated mobile application, developed using Kotlin and Java in Android Studio, creates a virtual map and generates optimal paths between beacon devices for efficient navigation. Additionally, cloud services, specifically AWS IoT Core, are employed for device management and positioning data storage, enhancing system functionality and reliability
